# Manage Deploys

Learn how to manage, monitor, and control your deployments.

## Deployment Dashboard

Access your deployment dashboard to:
- View all deployments
- Check deployment status
- View build logs
- Manage environment variables

## Deployment Actions

### Promote
Promote a preview to production.

### Rollback
Quickly rollback to a previous deployment.

### Delete
Remove old deployments to save resources.

### Clone
Create a new deployment based on an existing one.

## Monitoring

Track key metrics:
- Build time
- Deploy time
- Success rate
- Error logs

## Troubleshooting

Common issues and solutions:
- Build failures
- Timeout errors
- Configuration problems
- Environment variable issues

# Deploy Previews

Deploy previews let you see changes before they go to production.

## What are Deploy Previews?

Deploy previews are temporary environments created automatically for pull requests and branches.

## Features

- Unique URL for each preview
- Automatic creation on PR
- Comment notifications
- Linked to commits

## Using Previews

1. Create a pull request
2. Wait for preview to build
3. Click the preview URL
4. Review changes
5. Merge when ready

## Preview Environments

Previews include:
- Full application functionality
- Database connections (staging)
- Environment variables
- API integrations

# Deploy Overview

Learn about the various deployment options available for your Angular applications.

## Deployment Types

- Production deploys
- Preview deploys
- Branch deploys
- Development deploys

## Key Features

- Automatic deployments on push
- Preview URLs for every branch
- Rollback capabilities
- Custom domains

## Getting Started

Choose the deployment strategy that best fits your workflow and start deploying!

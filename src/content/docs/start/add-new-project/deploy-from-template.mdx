# Deploy from a Template

Use our pre-built templates to quickly start a new project with best practices already configured.

## Available Templates

- Starter Template
- Dashboard Template
- E-commerce Template
- Blog Template

## How to Use

1. Choose a template from the gallery
2. Click "Deploy"
3. Configure your project settings
4. Start building!

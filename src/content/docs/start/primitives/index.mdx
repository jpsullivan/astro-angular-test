# Primitives

Understand the fundamental building blocks (primitives) that power Angular Components.

## What are Primitives?

Primitives are the core, low-level components that serve as building blocks for more complex UI elements.

## Core Concepts

- Component primitives
- Directive primitives
- Service primitives
- Pipe primitives

## Why Primitives Matter

Using primitives gives you maximum flexibility and control over your application's behavior and appearance.

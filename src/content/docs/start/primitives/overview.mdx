# Primitives Overview

A comprehensive look at all available primitives in Angular Components.

## Component Primitives

Base components that can be extended and customized.

### Button Primitive
The foundation for all button-like interactions.

### Input Primitive
Base for all form input elements.

### Container Primitive
Layout and structure foundation.

## Directive Primitives

Low-level directives for common behaviors:

- Click handling
- Hover states
- Focus management
- Accessibility features

## Best Practices

- Always start with primitives when building custom components
- Don't reinvent the wheel
- Follow composition patterns

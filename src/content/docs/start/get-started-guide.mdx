# Get Started Guide

Follow this comprehensive guide to set up your first Angular Components project.

## Prerequisites

- Node.js 18 or higher
- npm or yarn package manager
- Basic knowledge of Angular

## Installation

```bash
npm install @your-library/angular-components
```
